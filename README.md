## How to run
```
./fpgrars-x86_64-pc-windows-gnu.exe src\main.asm
```
## How to play
```
[A], [W], [S], [D] - player move
[1], [2], [E] - menu interactions

Cheat codes: (on first menu screen or on gameover screen)

[X] - first phase
[Y] - second phase
[Z] - third phase
[E] - endgame screen
[L] - level complete screen
```
